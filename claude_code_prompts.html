Claude Code Prompt Guide
命星 — VS Code 개발 프롬프트 모음
각 단계별로 Claude Code에 붙여넣기 하세요
시작 전 필수 — 프로젝트 폴더 안에 CLAUDE.md 파일을 먼저 넣어두세요.
Claude Code는 이 파일을 자동으로 읽어 프로젝트 전체 구조를 이해합니다. 매번 설명할 필요가 없어집니다.
VS Code 터미널에서 claude 명령어로 시작 → /init 입력하면 CLAUDE.md가 생성됩니다.
Phase 0
프로젝트 초기화 — 가장 먼저 실행
프로젝트 뼈대 생성
최초 1회
CLAUDE.md 파일을 읽고 命星(명성) 사주 웹앱 프로젝트의 Spring Boot 기반 Maven 프로젝트 뼈대를 생성해줘. 아래 조건을 지켜줘: 1. Spring Boot 3.x + Java 17 2. pom.xml에 필요한 의존성 포함: - spring-boot-starter-web - spring-boot-starter-thymeleaf - mybatis-spring-boot-starter - mysql-connector-java - lombok - spring-boot-starter-security (기본만) 3. application.properties에 DB 연결 설정 (localhost, DB명: myeongseong_db) 4. CLAUDE.md의 디렉토리 구조대로 패키지와 빈 파일들을 생성해줘 5. 각 파일에 한국어 주석으로 역할 설명 포함
DB 테이블 생성 SQL
최초 1회
CLAUDE.md의 DB 테이블 구조를 바탕으로 MySQL용 DDL SQL 파일을 생성해줘. 요구사항: 1. 파일명: src/main/resources/sql/schema.sql 2. 테이블 순서: users → saju_info → manjuryok → daily_fortune → saju_report 3. 각 테이블에 적절한 INDEX 추가 (user_id, fortune_date 등 조회에 쓰이는 컬럼) 4. 외래키 제약조건 포함 5. CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci 설정 6. 각 테이블 앞에 한국어 주석으로 용도 설명
Phase 1
만세력 + 사주 계산 로직
SajuService — 날짜→사주 계산
최초 1회
@SajuService.java 파일을 열어서 아래 기능을 구현해줘. 구현할 메서드: 1. calculateSaju(LocalDate birthDate, String birthTime, boolean isLunar) - 양력/음력 날짜를 받아 manjuryok 테이블에서 천간지지 조회 - 시주는 일간(日干) 기준으로 시간대별 계산 (자시~해시) - 결과를 SajuInfoVO에 담아 반환 2. getTodayGanji() - 오늘 날짜의 일진(천간지지) 반환 - manjuryok 테이블에서 오늘 날짜 조회 조건: - MyBatis ManjuryokMapper 사용 - 예외 처리 포함 (날짜 범위 벗어나면 명확한 에러 메시지) - 각 메서드마다 한국어 주석으로 동작 설명 - 시주 계산 로직은 전통 명리학 기준 (야자시 고려 선택사항)
만세력 샘플 데이터 INSERT
최초 1회
manjuryok 테이블에 넣을 샘플 데이터 SQL을 작성해줘. 조건: 1. 2024년 1월 1일 ~ 2025년 12월 31일 (2년치) 데이터 2. 각 날짜의 solar_date, year_gan, year_ji, month_gan, month_ji, day_gan, day_ji 포함 3. 실제 명리학 기준 천간지지 사용 (갑자년=2024년 기준) 4. 파일명: src/main/resources/sql/manjuryok_sample.sql 5. 100건씩 나눠서 INSERT 구문 작성 참고: 2024년은 甲辰년(갑진년), 2025년은 乙巳년(을사년)
Phase 2
Claude API 연동 — 핵심 기능
SajuPromptBuilder — 프롬프트 조립
핵심
@SajuPromptBuilder.java 파일에 프롬프트 조립 로직을 구현해줘. 이 클래스는 사주 데이터를 받아서 Claude API에 보낼 프롬프트 문자열을 만드는 역할이야. 메서드: buildDailyFortunePrompt(SajuInfoVO saju, String todayGan, String todayJi) 프롬프트 내용에 반드시 포함할 것: 1. 사용자 일간(日干) — 예: 甲木(갑목) 2. 사주팔자 8글자 전체 3. 오늘 일진 (천간지지) 4. 출력 형식을 명확히 지정 — 반드시 JSON만 반환, 다른 텍스트 없이 5. JSON 키: total_luck, work_luck, love_luck, health_luck, caution, lucky_color, lucky_number, lucky_item 6. 각 항목 2~3문장, 친근하고 따뜻한 말투, 부정적 내용은 완곡하게 추가 요구사항: - @Component 어노테이션 적용 - 프롬프트 템플릿은 상수로 분리 - 한국어 주석 포함
ClaudeApiService — API 호출
핵심
@ClaudeApiService.java 파일에 Claude API 호출 로직을 구현해줘. CLAUDE.md의 API 호출 방식을 참고해서 구현. 구현 내용: 1. callFortuneApi(String prompt) 메서드 - RestTemplate으로 POST /v1/messages 호출 - 요청 헤더: x-api-key, anthropic-version, content-type - 요청 바디: model(claude-sonnet-4-5), max_tokens(1500), messages - 응답에서 content[0].text 추출 - Jackson으로 FortuneResponseVO에 파싱해서 반환 2. 예외 처리: - API 호출 실패 시 log.error 출력 - JSON 파싱 실패 시 기본값 FortuneResponseVO 반환 (fallback) - 타임아웃 설정 (30초) 3. API 키는 @Value("${claude.api.key}")로 주입 4. application.properties에 claude.api.key=여기에_키_입력 항목 추가 모든 메서드에 한국어 주석 필수.
FortuneService — 운세 생성 총괄
핵심
@FortuneService.java 파일에 운세 생성 전체 흐름을 구현해줘. 메서드 2개: 1. generateTodayFortune(Long userId) - SajuMapper로 사용자 사주 조회 - SajuService.getTodayGanji()로 오늘 일진 조회 - SajuPromptBuilder로 프롬프트 생성 - ClaudeApiService로 Claude API 호출 - 결과를 daily_fortune 테이블에 저장 - @Transactional 적용 2. getTodayFortune(Long userId) - daily_fortune에서 오늘 날짜로 조회 - 없으면 generateTodayFortune 호출 후 반환 - 있으면 DB에서 바로 반환 (API 재호출 방지) 비회원 처리: - generateGuestFortune(SajuInfoVO sajuInfo) 메서드 추가 - DB 저장 없이 API 호출 결과만 반환 한국어 주석 필수.
Phase 3
프론트엔드 — 화면 개발
랜딩 페이지 (index.html)
@templates/index.html 파일을 Thymeleaf 기반으로 만들어줘. 디자인 컨셉: 신비롭고 고급스러운 프리미엄 느낌 - 배경: 검정 계열 (#0F0E0C) - 포인트 색상: 골드 (#C9A84C) - 폰트: Noto Serif KR (제목), Noto Sans KR (본문) 포함할 요소: 1. 상단 로고 — 命星 한자 + 브랜드명 2. 메인 카피 — "당신의 별이 말하는 오늘" 같은 감성적 문구 3. 서브 카피 — 서비스 한 줄 소개 4. CTA 버튼 2개: - "지금 내 운세 보기" → /saju/input (비회원 체험) - "로그인" → /user/login 5. 하단 — 서비스 특징 3가지 간략 소개 CSS는 style.css 파일로 분리, JS는 필요한 경우만 반응형 (모바일 우선, max-width: 480px 기준)
운세 결과 화면 (today.html)
@templates/fortune/today.html 파일을 만들어줘. 이 화면은 Claude AI가 생성한 운세 결과를 보여주는 메인 화면이야. Thymeleaf 변수: ${fortune} (DailyFortuneVO 객체) 화면 구성: 1. 상단: 날짜 + 사용자 일간 표시 (예: "甲木 일간의 2025년 1월 15일") 2. 총운 카드 — 가장 크게, 오늘의 핵심 메시지 3. 운세 그리드 (2×2): - 직업·재물운 / 애정운 / 건강운 / 조심할 것 4. 행운 아이템 섹션: - 행운의 색상 (색상 원 + 이름) - 행운의 숫자 (크게 표시) - 행운의 물건 (아이콘 + 텍스트) 5. 비회원인 경우: "운세를 저장하고 매일 받아보세요" 배너 + 가입 버튼 디자인: 검정 배경, 골드 포인트, 카드마다 subtle한 테두리 로딩 중 스피너 애니메이션 포함
Phase 4
스케줄러 — 자동 운세 생성
SchedulerService — 매일 자정 자동 실행
@SchedulerService.java 파일에 자동 운세 생성 스케줄러를 구현해줘. 구현 내용: 1. generateAllUsersFortune() 메서드 - @Scheduled(cron = "0 0 0 * * *") — 매일 자정 실행 - users 테이블에서 활성 회원 전체 조회 - 각 회원마다 FortuneService.generateTodayFortune() 호출 - 실패한 회원은 건너뛰고 다음 진행 (전체 중단 방지) - 처리 결과 로그 출력 (성공 N명, 실패 N명) 2. Application 클래스에 @EnableScheduling 추가 필요 — 알려줘 3. 배치 처리 최적화: - 회원 수가 많을 경우를 대비해 100명씩 나눠서 처리 - 각 호출 사이 1초 간격 (API rate limit 방지) 주의사항: - Claude API 비용 최소화를 위해 이미 오늘 운세가 있으면 건너뜀 - 에러 로그 상세 출력 - 한국어 주석 필수

Setup
Claude Code 설치 순서
01
Node.js 설치

Node.js 18 이상 설치 필요. node --version 으로 확인.

02
Claude Code CLI 설치

터미널에서 npm install -g @anthropic-ai/claude-code 실행

03
VS Code Extension 설치

Extensions (Ctrl+Shift+X) → Claude Code 검색 → Anthropic 제작 확장 설치

04
CLAUDE.md 프로젝트 폴더에 복사

다운로드한 CLAUDE.md 파일을 프로젝트 루트(pom.xml과 같은 위치)에 넣기

05
VS Code에서 시작

프로젝트 폴더 열기 → 파일 하나 열기 → 우측 상단 번개(Spark) 아이콘 클릭 → Claude Code 패널 열림

06
첫 번째 프롬프트 입력
